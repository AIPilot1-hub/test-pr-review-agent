# test_bench

This project is a Python application designed to demonstrate a modular structure with various functionalities encapsulated in separate modules.

## Project Structure

- **main.py**: Entry point of the application.
- **module_a.py**: Contains specific features related to functionality A.
- **module_b.py**: Contains additional features related to functionality B.
- **security.py**: Manages security-related functionalities such as authentication and authorization.
- **utils.py**: Provides utility functions and constants used throughout the application.

## Installation

To install the necessary dependencies, run:

```
pip install -r requirements.txt
```

## Usage

To run the application, execute the following command:

```
python main.py
```
